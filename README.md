# Valorant Theme

A collection of VS Code themes inspired by the agents of Valorant.

## Available Themes

*   **Valorant: Wind (Jett)** - Inspired by Jett's cool blue and white aesthetic.
*   **Valorant: Blaze (Phoenix)** - Inspired by Phoenix's fiery orange and dark grey style.
*   **Valorant: Jade (Sage)** - Inspired by Sage's calming mint and jade green colors.
*   **Valorant: Empress (Reyna)** - Inspired by Reyna's dark purple and magenta vibe.
*   **Valorant: Shadow (Omen)** - Inspired by Omen's mysterious cyan and purple tones.
*   **Valorant: Toxic (Viper)** - Inspired by Viper's poisonous green and dark aesthetic.

## Installation

1.  Open Chrome or your browser of choice.
2.  Download the `.vsix` file.
3.  Open VS Code.
4.  Go to Extensions -> "..." -> Install from VSIX...
5.  Select the downloaded file.
6.  Go to File -> Preferences -> Color Theme and select one of the Valorant themes.
